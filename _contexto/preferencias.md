# Preferências de Comunicação — Spellbound Marketing

## Tom de voz

Arquétipo **Sábio Parceiro**: autoridade com proximidade. Ensina sem arrogância, provoca reflexão, simplifica o complexo, convida à ação estratégica.

Idioma principal dos materiais: **inglês** (mercado NL). Português para comunicação interna e operação.

## Estilo geral

- 1ª pessoa do plural ("vamos", "podemos") alternando com 2ª ("você/you") para proximidade
- Frases curtas para impacto + blocos curtos explicativos
- Estrutura favorita: **Pergunta → Contexto → Insight → Ação/CTA**
- Voz ativa e direta. Sem gerundismos longos
- Dados + interpretação + próximo passo. Sempre mostrar "o que isso significa pra você"
- O sparkle ✦ é o único elemento decorativo — usar no lugar de emojis genéricos

## O que evitar

- Jargão técnico sem explicação
- Promessas mágicas ("explode vendas em 24h", "garantido", "fácil")
- Tom escolar ou distante ("O usuário deve…")
- Hype vazio — a Spellbound conquista por **método e clareza**, não por atalhos
- Superlativos vagos ("o melhor", "revolucionário")

## Substituições de palavras

| Evitar | Usar |
|--------|------|
| problema | challenge / opportunity |
| barato | accessible / economical |
| desconto | exclusive offer |
| fácil | practical / strategic |
| rápido | sustainable / structured |
| simples | well-structured / optimized |
| garantido | proven / tested |
| grátis | complimentary |
| melhor | superior |

## Checklist antes de finalizar qualquer texto

- O texto educa e direciona?
- Há pergunta reflexiva ou insight?
- Dados têm fonte confiável?
- Complexidade traduzida em ação prática?
- CTA alinhado à jornada (descoberta, consideração, conversão)?
